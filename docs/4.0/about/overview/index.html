Team

Bootstrap is maintained by Federico (@FezVrasta) Zivolo on GitHub. I’m actively looking to grow the team and would love to hear from you if you’re excited about CSS at scale, writing and maintaining vanilla JavaScript plugins, and improving build tooling processes for frontend code.

Get involved

Get involved with Material Design for Bootstrap development by opening an issue or submitting a pull request. Read our contributing guidelines for information on how we develop.
